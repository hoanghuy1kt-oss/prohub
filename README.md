## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The app will be available at `http://localhost:5173` (or another port if 5173 is in use).

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
